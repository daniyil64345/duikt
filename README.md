Telegram-бот для управління реальним інтернет-магазином з підтримкою бази даних та інтерактивних клавіатур. Бот дозволяє адміністраторам керувати товарами, обробляти замовлення, а користувачам — зручно робити покупки прямо в Telegram. Використовується SQLite для зберігання даних та підтримується асинхронна робота через asyncio.

Можливості:

1) Каталог товарів з можливістю перегляду та покупки

2) Обробка замовлень та кошик покупок

3) Адміністраторська панель для управління магазином

4) Інтерактивні клавіатури для зручної навігації

5) База даних SQLite для зберігання товарів та замовлень

6) Middleware для обробки запитів

7) Фільтри для різних типів користувачів

Технології

1) Python 3.8+	Основна мова програмування
2) aiogram / python-telegram-bot	Асинхронна робота з Telegram Bot API
3) SQLite	Локальна база даних
4) asyncio	Асинхронне виконання операцій


Встановлення та запуск

Клонування репозиторію:

`git clone https://github.com/daniyil64345/tg_bot.git`

`cd tg_bot`


Встановлення залежностей:

`pip install -r requirements.txt`


Створення файлу `.env` з токеном бота та параметрами адміністратора:

`BOT_TOKEN=your_telegram_bot_token`
`ADMIN_IDS=123456789,987654321`
`DATABASE_URL=sqlite:///true_detective.db`


Ініціалізація бази даних:

База даних створюється автоматично при першому запуску.


Запуск бота:

`python bot_main.py`

Як це працює

1) Ініціалізація: При запуску бот підключається до бази даних та реєструє всі обробники.

2) Обробка команд: Користувач надсилає команду → фільтри перевіряють права → middleware обробляє запит → обробник виконує логіку.

3) Робота з БД: Всі дані зберігаються в SQLite, що забезпечує швидкий доступ та надійність.

4) Інтерактивність: Кастомні клавіатури роблять навігацію інтуїтивною та зручною.

5) Асинхронність: Бот може обробляти багато запитів одночасно завдяки asyncio.
